Branching

You may want to show a user different pages, depending on how they've answered a question.

You can make this happen by including value="redirect(<url>)" in your radio button values. For example:

<div class="multiple-choice"> <input id="workStatusEmp" type="radio" name="work-status" value="redirect(branch/employed)"> <label for="workStatusEmp">Employed</label> </div> <div class="multiple-choice"> <input id="workStatusSelf" type="radio" name="work-status" value="redirect(branch/self-employed),Self-Employed"> <label for="workStatusSelf">Self-employed</label> </div> <div class="multiple-choice"> <input id="workStatusDir" type="radio" name="work-status" value="redirect(branch/director)"> <label for="workStatusDir">Director of a limited company</label> </div>

If you want to pass a value as well as the redirect instruction, put the value after a comma within the input value. For example: value="redirect(<url>),<value>"
