Branching

You may want to show a user different pages, depending on how they've answered a question.

Simple method (radio buttons only)

If you're using radio buttons, you can set the page to redirect to in the value of the input element.

In this example the code is:

<input type="radio" name="over18" value="redirect(/docs/examples/branching/over-18)">

Are you 18 or over?

Are you 18 or over?

Advanced method

You can write some 'route' code to branch on any kind of input.

You can see the code for this in /docs/documentation_routes.js

Enter 0, 5 or 20 to see the branching pages
