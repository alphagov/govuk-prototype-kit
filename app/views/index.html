About this prototype

This prototype explores different ways for users to navigate Manuals and HTML Publications.

HTML Publications Prototype

We have made a few changes to the layout of HTML Publications. These changes were made in order to address certain known issues with the layout, and to make the layout consistent with the work being done on the Short Text templates.

Manuals Prototype

  • Prototype B - Chapter Contents List
