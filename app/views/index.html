This kit lets you rapidly create HTML prototypes of internal Home Office services.

It contains all you need to prototype anything from simple static pages to complex, data-driven transactions. It's based on the GOV.UK Prototype Kit, which:

  • is styled for internal Home Office applications
  • has additional Home Office components

About this page

We recommend you add links to your prototype to this page, rather than delete it. You'll find this page in the kit at '/app/views/index.html'.

You can change the service name by editing the file '/app/config.js'. This will appear in the header.

Use this prototyping kit

We've extended the GOV.UK design system for additional use cases and users, including internal services.

They have been developed by the Home Office research and design teams to help develop and create consistent, usable services.

Learn more about the Home Office design system including example components and guidance on usage.

We recommend you don't change the Home Office styles, found in 'app/assets/sass/home-office'. If you find a problem or a user need to change a pattern, contribute to the design system which will help us to improve it so that everyone can benefit.
