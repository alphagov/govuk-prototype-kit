This kit lets you rapidly create HTML prototypes of GOV.UK and internal Home Office services.

It contains all you need to prototype anything from simple static pages to complex, data-driven transactions.

About this page

We recommend you add links to your prototype to this page, rather than delete it. You'll find this page in the kit at '/app/views/index.html'.

You can change the service name by editing the file '/app/config.js'.

Using the Home Office design patterns

These patterns extend the GOV.UK design patterns for additional use cases and users, including internal services.

They have been developed by the Home Office research and design teams to help develop and create consistent, usable services.

Learn more about the Home Office design guide and patterns library including example components and guidance on usage.

Styles specific to these patterns are included in this kit. Take a look at the individual Sass files to understand the detail behind a pattern.

The individual pattern files are imported into the main application.scss file, ready to use.

We do not recommend you edit these files. If you find a problem or a user need to change a pattern, please contribute to the patterns library which will help us to improve it so that everyone can benefit.
