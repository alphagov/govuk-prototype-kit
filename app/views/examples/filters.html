Using built-in and custom filters in templates

Nunjucks ships with a set of builtin filters documented on it's website. For example, you can pipe a string to the 'upper' filter as here:

{% raw %}{{ "Hello World" | upper }}{% endraw %} will output: {{ "Hello World" | upper }}

You can also add your own custom filters to: /lib/filters.js for example this padZeros filter.

{% raw %}{{ 3 | padZeros(2) }}{% endraw %} will output: {{ 3 | padZeros(2) }}
