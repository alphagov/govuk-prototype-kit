Logging data

You can send data to the browser console using the "log" filter in your template:

{{ users | log }}

{% raw %}{{ users | log }}{% endraw %}

Open the browser console to see the data in this example. In Chrome, press the cmd + alt + j keys together to open the console.

This can be useful when using complex data in your prototype, to check it looks correct.
