Background

There are many ways to implement show/hide functionality. GOV.UK currently uses an implementation of the details element to do this.

Questions I have include:

  • Are voice recognition users able to open and close them?
  • How does it work in screen readers. Is the content always available - or does a screen reader user have to open the widget to access the summary information. If they do state (open and closes) communicated?
  • What browsers are supported? What is the fallback for old browsers?
