Background

The service manual has a design pattern for collecting memorable dates. There is a code example for this in the GOV.UK Elements. The code example uses a fieldset to group the inputs, with a legend for the title and paragraph (with no wai-aria) for the hint. To prevent users from inputting the wrong information the input attributes pattern, min and max are used. Is this the best implementation for screen readers and other assitive technologies?
